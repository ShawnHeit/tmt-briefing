周度复盘 Weekly Review
周期：04/21-04/28 · 来源：近 7 日每日汇总（手动补充 / ZSXQ / AceCamp / kanzhiqiu）
AI 主线再确认，主升浪阶段方向和仓位大于选股
04/21-04/28 的 daily 信息继续围绕 AI 算力、云 Capex、光模块、存储与北美互联网展开；从外资研报和手动补充材料看，AI 投资主线从“GPU 单点”扩散到 CPU/TPU/ASIC、网络、存储、电源散热和光互联，主升阶段方向和仓位仍大于个股精挑。
阅读顺序：先看事实 灰色 chip = source / 原文 再看推演和建议
AI ASIC / TPU

现状及边际变化：Citi 单独梳理 Google TPU v6/v7/v8 路线，Google Cloud 另有 TPU v8t/v8i 技术深度材料，说明自研 TPU 已进入多代路线图跟踪 （Citi TPU，04/28）（Google Cloud，04/28）；JPM DeepSeek V4 把模型升级解释为行业顺风而非零和冲击，说明市场仍在讨论“效率提升到底削减还是放大推理算力需求” （JPM DeepSeek V4，04/28）。

推演：自研 ASIC/TPU 不会简单“替代 GPU 价值量”，而是把 AI 算力拆成更细的训练、prefill、decode、调度和网络层，放大非 GPU 服务器、互连、存储和供电需求。Google/AWS/Meta 自研芯片越快放量，越会推动市场从“只买 GPU beta”转向“买 ASIC 机架配套和推理控制面”。

建议：验证链看 Google TPU v7/v8 出货节奏、AWS Trainium/Inferentia 和 Meta MTIA 的 rack-scale 配套、以及 BMC/SerDes/Retimer/先进封装测试订单；标的上关注 Google TPU 链和 ASIC 机架配套环节。

CPU / x86 vs Arm

现状及边际变化：Agentic AI 把 CPU 从普通服务器配套件变成推理调度/工作流编排的控制平面，MS 估算 2030 年会带来 325-600 亿美元增量 CPU TAM，且 CPU:GPU 集群比例从 1:12 走向 1:2+ （MS Agent 大报告，04/28）；AceCamp 光架构纪要也提到 TPU 光模块场景里 CPU:GPU 配比从 1:8-12 走向 1:4 甚至更低 （AceCamp Steve Song，04/18）；BofA 单独讨论 x86 vs Arm vs ASIC 路线，Intel 同期被 Citi 上调至买入，理由是 AI CPU 周期 + Terafab 代工机会 （BofA CPU，04/28）（Citi Intel，04/24）。

推演：AI rack 的 CPU attach 上升不是短期交易噪音，而是 agentic workflow 把 CPU 从“喂 GPU 的管理节点”升级为“多步骤推理调度 + memory/storage orchestration 的控制面”。结果会是通用 CPU 需求从传统云服务器周期里独立出来，Intel/AMD/Arm 都受益，但份额会重新分配：x86 先吃存量数据中心兼容性，Arm/自研 CPU 在 hyperscaler 内部继续提升，ASIC/TPU 反而放大对 CPU 控制面的需求。

建议：AI 研究显示应重点看三条验证链：CPU:GPU attach 是否继续下探到 1:4/1:2+；Intel/AMD 是否在 AI server 订单里明确披露 CPU 拉动；Arm/Graviton/Muse 这类自研 CPU 是否扩大内部份额。标的上，海外看 Intel 反转、AMD EPYC、Arm 生态；A 股映射看澜起、信骅 BMC、服务器电源和内存接口链。

云 Capex / AI 服务器 ODM

现状及边际变化：海外大厂 2027 年 Capex 预期被上修 40%+，AI 服务器/算力维持高景气 （Capex +40%，04/28）；高盛上调 AI 机架/服务器 TAM，并给出 2028 年首次预测 （GS AI rack TAM，04/28）；Celestica 被单列为 AI 服务器/网络代工景气持续标的，鸿海也有 AI 服务器订单、墨西哥/美国产能和毛利率更新 （Celestica，04/28）（GFHK Hon Hai，04/28）。

推演：2027 Capex 上修会先在 rack 级交付、ODM 订单能见度和供电散热 BOM 上兑现，而不是先体现在所有 AI 应用公司收入里。AI 服务器从单机出货进入机架交付后，Celestica/鸿海/工业富联这类整机与网络代工的定价权会强于纯组装周期。

建议：验证链看云厂商 2027 Capex 指引、rack backlog、墨西哥/美国产能和 book-to-bill；标的优先跟踪工业富联/鸿海、广达/纬颖/Celestica、Vertiv/英维克/欧陆通，算力租赁只参与长约质量和资产负债表清晰的公司。

800G / 1.6T 光模块

现状及边际变化：野村光模块报告直接覆盖 800G / 1.6T 出货预期、CW 光源和 CPO 进展 （Nomura 光模块，04/28）；中际旭创产业链调研把索尔斯、旭创、新易盛、物料和海外进度放在同一份材料里，说明市场跟踪已从单一龙头业绩扩展到供应链交付 （光模块产业链调研，04/28）；AceCamp 当日专家会也把“光模块与 CPO 电源管理/高速前端规格升级”列为议题 （AceCamp Bryce Wang，04/28）。

推演：光模块行情正在从“800G 放量”进入“1.6T 前移 + CW 光源/EML/物料瓶颈重定价”阶段。龙头业绩仍是底座，但真正的弹性会扩散到具备海外客户认证、物料卡位和高速前端能力的二线与上游组件。

建议：验证链看 1.6T 送样/量产节奏、CW 光源供应、海外客户认证和物料交期；标的龙头继续看中际旭创/新易盛/天孚，弹性看联特/剑桥/汇绿，组件链看 EML/CW 光源、测试设备和高速前端。

光器件 / OCS / CPO

现状及边际变化：光学专题明确覆盖光器件、EML、CW、800G & 1.6T 进展 （光学行业专题，04/28）；JPM 硬件与网络周报继续跟踪交换芯片、ODM 和网卡需求更新 （JPM Hardware & Networking，04/28）；BofA 康宁材料指向光纤需求回暖和玻璃基板/特种玻璃 ramp （BofA 康宁，04/28）。

推演：OCS/CPO 不会立刻替代传统光模块，但会先改变高速前端、光器件和光材料的估值锚。只要客户验证持续推进，市场会把“下一代架构期权”提前给到 EML/CW、硅光、光纤/玻璃和高速前端，而不是等到收入完全兑现。

建议：验证链看 OCS/CPO 客户测试、光网络整机订单、康宁光纤/玻璃基板 ramp 和高速前端规格升级；标的跟踪腾景/光库/德科立、源杰/长光华芯、康宁/云南锗业/AXT 等光材料映射。

PCB / CCL / 玻璃基板

现状及边际变化：Bernstein AI Server Pulse 以“Chasing Scarcity”为主题，明确关注 HBM、CCL 和 substrate 基板稀缺性 （Bernstein AI Server Pulse，04/22）；Jefferies PCB/CCL 把 AI 红利受益标的落到 WUS/Delton/Sytech，新覆盖本身就是资金开始细拆材料链的信号 （Jefferies PCB/CCL，04/20）；玻璃基板专题覆盖先进封装中介层、TGV、设备与材料受益方 （玻璃基板，04/28）。

推演：AI 服务器材料链会从“GPU/HBM 稀缺”扩散到“高速 CCL + substrate + 玻璃基板”的二阶稀缺，且市场会优先给具备涨价能力、良率爬坡和客户认证的公司估值溢价。玻璃基板会作为先进封装中介层期权被交易，但兑现节奏慢于 CCL/PCB。

建议：验证链看 CCL/PCB 涨价、TGV 工艺进展、玻璃基板客户导入和良率；标的关注生益科技、沪电/深南、胜宏、东山精密、三环/三星电机映射，玻璃基板只参与有设备或客户导入事实的公司。

存储 / NAND / HDD

现状及边际变化：Sandisk 在 04/27 ZSXQ 日报中与 NVIDIA 同帖出现，04/28 手动补充又单列“闪迪-上调”，说明 NAND 纯供应商被连续跟踪 （Sandisk 上调，04/28）；STX/WDC 报告标题为“Clear path to higher earnings; Raising PO”，HDD 侧也有明确盈利上修路径 （STX/WDC，04/27）；SK 海力士 1Q26 材料写到 ASP 强势可持续、HBM3E/HBM4 路线确认 （SK Hynix，04/28）。

推演：存储会从“周期修复”切换成“AI 数据基础设施重估”，NAND、HDD 和 HBM 的价格弹性会分别对应推理缓存、冷数据归档和高端训练/推理。Sandisk/STX/WDC 这种纯度更高的供应商会比综合存储公司更先被定价。

建议：验证链看 NAND/HDD ASP、HBM3E/HBM4 长协、云厂商存储 capex 和库存；美股跟 Sandisk、STX/WDC、美光，韩股跟 SK Hynix，A 股跟兆易、江波龙、澜起和存储设备/材料映射。

国产推理算力

现状及边际变化：AceCamp 火山调研主题直接落到 AI 推理算力供需变化、昇腾/寒武纪/海光采购规划，以及与阿里云竞争变化 （AceCamp Jay Dong，04/28）；JPM DeepSeek V4 报告认为 DeepSeek V4 是行业顺风而非零和冲击，继续维持智谱/MiniMax 增持 （JPM DeepSeek V4，04/28）；DeepSeek V4 解读材料明确把推理 token 升量与国内云厂商/数据中心/算力链传导放在一起讨论 （DeepSeek V4 解读，04/28）。

推演：国内推理算力会从“国产替代叙事”进入“谁拿到真实云厂商采购和推理 token 放量”的验证阶段。DeepSeek V4 这类模型升级如果继续降低应用门槛，反而会把更多需求推向推理侧，国产卡、云厂商数据中心和配套电源/液冷/网络会先受益。

建议：验证链只看云端推理：昇腾/寒武纪/海光采购规划、阿里云竞争变化、DeepSeek V4 带来的推理 token 需求、以及国产卡在数据中心真实部署节奏；标的跟踪寒武纪、海光、昇腾生态和国产算力配套链。
